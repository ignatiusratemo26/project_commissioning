# Project Commissioning

This project is a React application built with Vite. It includes user authentication, project management, and an admin dashboard.

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/ignatiusratemo26/project_commission.git
    cd project_commissioning
    ```

2. Install the dependencies:
    ```sh
    npm install
    ```

## Usage

1. Start the development server:
    ```sh
    npm run dev
    ```

2. Open your browser and navigate to `http://localhost:3000`.
